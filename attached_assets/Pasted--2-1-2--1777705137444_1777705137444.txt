현재 운영 중인 “초능력자” 봇의 게임/포인트/유물 시스템을 개편하고자 합니다.

초능력자는 현재 메인게임 2개만 존재합니다.

1. 초성퀴즈
2. 훈민정음

현재 문제점은 다음과 같습니다.

- 포인트를 획득할 수는 있지만, 포인트를 사용할 수 있는 명확한 시스템이 부족합니다.
- 초성퀴즈와 훈민정음은 기존 유저에게는 괜찮지만, 신규 유저에게는 다소 불친절합니다.
- 유물 시스템을 만들고 있었지만, 기존 방향은 단일 유물을 1강~100강까지 강화하는 구조에 가까워 장기적으로 단조로울 수 있습니다.
- 따라서 게임 구조를 “메인게임 3개 + 서브게임 2개 + 유물 시스템”으로 재편하려 합니다.

아래 요구사항에 맞춰 전체 시스템을 설계 및 구현 가능한 작업 단위로 정리해주세요.

---

# 1. 전체 구조 개편 방향

초능력자의 핵심 구조는 다음과 같이 변경합니다.

메인게임:
- 언제든 플레이 가능
- 포인트를 획득하는 기본 수단
- 서브게임 입장권 또는 유물 관련 보상을 낮은 확률로 획득 가능

서브게임:
- 메인게임에서 얻은 도전권이 있어야 플레이 가능
- 포인트를 증폭하거나 잃을 수 있는 도박성 미니게임
- 포인트 인플레이션 방지를 위해 무제한 반복 플레이는 불가능해야 함

유물 시스템:
- 포인트를 사용하는 핵심 성장 시스템
- 단일 유물을 1강~100강까지 일직선으로 키우는 기존 구조는 폐기
- 대신 “메인 유물 1개 + 보관함 유물 여러 개 + 재료 유물 소모 + 등급/강화/각성” 구조로 변경

---

# 2. 기존 유물 시스템에서 버릴 것

기존 유물 시스템 중 아래 요소는 폐기하거나 대체합니다.

## 폐기할 요소

1. 단일 유물 1개를 1강~100강까지 단순 강화하는 구조
- 기존 방식은 성장감은 있으나, 수집/보관/조합의 재미가 부족함
- 앞으로는 유물 하나만 계속 강화하는 방식이 아니라 여러 유물을 얻고, 보관하고, 재료로 사용하고, 메인 유물을 키우는 구조로 변경

2. 강화 수치만으로 유물의 가치를 결정하는 구조
- 기존에는 “몇 강인가”가 거의 전부였지만, 앞으로는 등급, 레벨, 강화, 각성 옵션, 보관함 효과가 함께 중요해야 함

3. 모든 유물이 동일한 성장 방식으로 강화되는 구조
- 유물마다 등급, 효과, 성장 가치가 다르게 설계되어야 함

4. 유물 이미지에 맞춰 1강~100강 이름을 모두 다르게 붙이는 방식
- 기존에는 하나의 이미지에 대해 1강~100강 이름을 붙이는 방식이었으나, 앞으로는 등급과 상태에 따라 이름이 변화하는 구조가 더 적합함
- 예: 낡은 나침반 → 고대의 나침반 → 각성한 나침반 → 초월한 나침반

5. 포인트 사용처가 오직 강화뿐인 구조
- 앞으로 포인트는 강화, 승급, 각성, 보관함 확장, 서브게임 베팅 등에 사용되어야 함

## 유지해도 되는 요소

1. 유물 강화 개념
- 단, 기존의 단순 1~100강 구조가 아니라, 유물 레벨/강화/등급/각성 구조 안에 포함

2. 유물 이미지 사용
- 이미지 자체는 유지 가능
- 단, 같은 이미지를 사용하더라도 등급과 상태에 따라 이름과 효과가 달라지도록 설계

3. 포인트를 유물 성장에 사용하는 방향
- 이 방향은 유지
- 다만 포인트 소모처를 더 다양화해야 함

# 3. 메인게임 구성

최종 메인게임은 총 3개로 구성합니다.

1. 초성퀴즈
2. 훈민정음
3. 자모연성

기존 초성퀴즈와 훈민정음은 유지하되, 신규 유저 친화성을 높이기 위해 세 번째 메인게임으로 “자모연성”을 추가합니다.

# 4. 신규 메인게임: 자모연성

## 게임명

가칭: 자모연성

대체 이름 후보:
- 낱말연성
- 한글연금술
- 글자조합소
- 자모조합

우선 개발명은 “자모연성”으로 사용합니다.

## 기본 컨셉

사용자에게 자음과 모음을 제시하고, 해당 자모로 만들 수 있는 단어를 고르게 하는 게임입니다.

완전 자유입력형은 검증 난도가 높으므로, 초기 버전은 객관식으로 구현합니다.

## 기본 진행 방식

봇이 자모 목록과 선택지를 제시합니다.

예시:

지급된 자모:
ㄱ, ㄴ, ㅁ, ㅅ, ㅏ, ㅓ, ㅣ, ㅜ

문제:
위 자모로 만들 수 있는 단어는?

1. 가면
2. 도서관
3. 바나나
4. 컴퓨터

정답:
1. 가면

## 난이도 구조

난이도는 다음과 같이 나눕니다.

쉬움:
- 자모 목록 제공
- 4지선다 선택형
- 2글자 단어 중심

보통:
- 자모 목록 제공
- 4지선다 선택형
- 3글자 단어 포함

어려움:
- 자모 목록 제공
- 5지선다 또는 제한 시간 적용
- 받침 포함 단어, 3~4글자 단어 포함

추후 확장:
- 자유입력형
- 제한 시간 모드
- 긴 단어 보너스
- 오늘의 자모 보너스

## 보상 구조

자모연성 성공 시 포인트를 지급합니다.

추천 기본 보상:
- 쉬움 성공: 80P
- 보통 성공: 120P
- 어려움 성공: 180P

추가 보너스:
- 3글자 이상 단어 문제 성공 시 +30P
- 받침 포함 단어 문제 성공 시 +20P
- 연속 정답 3회 달성 시 +100P
- 하루 첫 자모연성 성공 시 +50P

서브게임 도전권 획득 확률:
- 기본 성공 시 5%
- 보통 이상 난이도 성공 시 8%
- 어려움 성공 시 12%
- 연속 정답 3회 달성 시 추가 1회 판정

획득 가능한 도전권:
- 운명권: 홀짝 플레이에 사용
- 예언권: 하이로우 카드 플레이에 사용

# 5. 서브게임 구성

서브게임은 총 2개로 구성합니다.

1. 운명 홀짝
2. 하이로우 카드

서브게임은 메인게임에서 얻은 도전권이 있어야만 플레이할 수 있습니다.

# 6. 서브게임 1: 운명 홀짝

## 게임명

가칭: 운명 홀짝

대체 이름 후보:
- 차원의 홀짝
- 운명의 동전
- 균열의 홀짝

우선 개발명은 “운명 홀짝”으로 사용합니다.

## 입장 조건

- 운명권 1장 필요
- 베팅 포인트 필요

## 베팅 가능 포인트

초기 추천값:
- 100P
- 300P
- 500P

추후 유저 평균 포인트에 따라 조정 가능하게 설정값으로 분리해주세요.

## 진행 방식

1. 사용자가 운명권 1장을 사용합니다.
2. 베팅 포인트를 선택합니다.
3. 홀 또는 짝을 선택합니다.
4. 봇이 랜덤 숫자를 생성합니다.
5. 홀짝 결과에 따라 성공/실패를 처리합니다.

## 결과

성공:
- 베팅 포인트 2배 지급

실패:
- 베팅 포인트 소멸

예시:
- 300P 베팅 성공 → 600P 획득
- 300P 베팅 실패 → 300P 차감

## 유물 효과 적용 주의

유물이 홀짝 성공 확률을 과도하게 올리면 포인트 인플레이션이 발생할 수 있으므로, 확률 증가 효과는 제한적으로만 허용합니다.

권장:
- 성공 확률 직접 증가 효과는 최대 +3%~5%까지만 허용
- 대신 실패 시 일부 환급, 하루 1회 보호, 추가 보너스 지급 같은 방식 권장

# 7. 서브게임 2: 하이로우 카드

## 게임명

가칭: 하이로우 카드

대체 이름 후보:
- 차원의 하이로우
- 운명 카드
- 예언자의 패
- 고대 카드

우선 개발명은 “하이로우 카드”로 사용합니다.

## 입장 조건

- 예언권 1장 필요
- 베팅 포인트 필요

## 베팅 가능 포인트

초기 추천값:
- 100P
- 300P
- 500P

## 진행 방식

1. 사용자가 예언권 1장을 사용합니다.
2. 베팅 포인트를 선택합니다.
3. 봇이 현재 카드를 공개합니다.
4. 사용자는 다음 카드가 현재 카드보다 높을지 낮을지 선택합니다.
5. 성공 시 다음 단계로 넘어갈 수 있습니다.
6. 사용자는 매 단계마다 “수령” 또는 “계속 도전”을 선택할 수 있습니다.
7. 실패 시 누적 보상은 소멸합니다.

## 카드 범위

- 카드 숫자: 1~13
- 현재 카드와 다음 카드가 같을 경우:
  - 초기 버전에서는 실패 처리
  - 추후 무승부 처리 옵션 추가 가능

## 연승 배율

초기 추천 배율:

1연승: 1.3배
2연승: 1.8배
3연승: 2.5배
4연승: 3.5배
5연승: 5.0배

최대 5연승까지만 허용합니다.

예시:
- 300P 베팅
- 1연승 성공 시 390P 수령 가능
- 2연승 성공 시 540P 수령 가능
- 3연승 성공 시 750P 수령 가능
- 실패 시 누적 보상 0P

## 경제 안정 장치

- 하이로우는 최대 5연승까지만 허용
- 도전권 없이는 플레이 불가
- 유물 효과로 배율을 과하게 올리지 않음
- 유물 효과는 보상 소폭 증가, 실패 보호, 도전권 획득률 증가 중심으로 설계

# 8. 유물 시스템 개편

유물 시스템은 기존의 단순 강화 시스템에서 다음 구조로 변경합니다.

## 핵심 구조

유저는 다음과 같은 유물 관련 데이터를 가집니다.

1. 메인 유물
- 대표로 장착하고 성장시키는 핵심 유물
- 1개만 선택 가능

2. 보관함 유물
- 여러 개 보유 가능
- 보관함에 있는 유물은 약한 패시브 효과를 제공할 수 있음

3. 재료 유물
- 메인 유물 강화, 레벨업, 승급, 각성에 사용
- 낮은 등급 유물도 재료로 의미를 가짐

---

# 9. 유물 등급

유물 등급은 다음 5단계로 구성합니다.

1. 일반
2. 희귀
3. 고대
4. 각성
5. 초월

기존에 고려했던 “일반, 고대, 각성, 초월” 구조에 “희귀”를 추가하여 성장 단계를 더 부드럽게 만듭니다.

## 등급별 의미

일반:
- 가장 기본적인 유물
- 주로 재료 또는 초반 보관함 효과용

희귀:
- 일반보다 높은 효과
- 초반 메인 유물 후보 가능

고대:
- 본격적인 성장 가치가 있는 유물
- 고유 효과가 뚜렷해지는 구간

각성:
- 특수 옵션이 열리는 등급
- 서브게임, 강화 보호, 보상 증가 등 특별한 효과 가능

초월:
- 최상위 등급
- 매우 낮은 확률로 획득하거나 장기 성장 끝에 도달

---

# 10. 유물 성장 구조

유물은 다음 요소를 가집니다.

1. 등급
2. 레벨
3. 강화 수치
4. 기본 효과
5. 각성 효과
6. 보관함 효과
7. 메인 장착 여부

## 레벨

- 유물 경험치를 통해 상승
- 재료 유물을 소모하여 경험치를 획득
- 레벨이 오르면 기본 효과 증가

## 강화

- 포인트와 재료를 사용해 강화
- 강화 수치가 오르면 효과 증가
- 강화 실패 가능 여부는 설정값으로 분리
- 초기 버전에서는 실패 없이 비용 증가형으로 구현하는 것을 권장

## 승급

유물은 일정 조건을 만족하면 다음 등급으로 승급할 수 있습니다.

예시:
- 일반 → 희귀
- 희귀 → 고대
- 고대 → 각성
- 각성 → 초월

승급에는 다음 재화가 필요합니다.
- 포인트
- 동일 등급 유물
- 특정 재료 유물
- 일정 레벨 조건

## 각성

각성 등급 이상부터 특수 옵션을 개방합니다.

예시:
- 하루 1회 서브게임 실패 보호
- 메인게임 포인트 획득량 증가
- 운명권 획득률 증가
- 예언권 획득률 증가
- 강화 비용 감소
- 유물 경험치 획득량 증가

---

# 11. 유물 효과 분류

유물 효과는 다음 네 계열로 분류합니다.

## 1. 메인게임 보상형

예시:
- 초성퀴즈 포인트 +3%
- 훈민정음 포인트 +3%
- 자모연성 포인트 +3%
- 모든 메인게임 포인트 +2%
- 연속 정답 보너스 +5%

## 2. 서브게임 기회형

예시:
- 운명권 획득률 +2%
- 예언권 획득률 +2%
- 하루 1회 운명권 추가 획득
- 메인게임 5연속 성공 시 추가 도전권 획득 확률 증가

## 3. 손실 완화형

예시:
- 홀짝 실패 시 5% 확률로 베팅금 30% 환급
- 하이로우 실패 시 3% 확률로 직전 단계 보상 일부 회수
- 하루 1회 서브게임 실패 보호
- 강화 비용 일부 반환

## 4. 성장 보조형

예시:
- 유물 경험치 획득량 +5%
- 강화 비용 -3%
- 보관함 슬롯 +1
- 일반 유물 획득률 +5%
- 고대 이상 유물 획득률 +1%

# 12. 보관함 시스템

유저는 유물 보관함을 가집니다.

## 기본 보관함 슬롯

초기값:
- 기본 10칸

확장:
- 포인트를 사용해 확장 가능
- 유물 효과로 보관함 슬롯 증가 가능

## 보관함 효과

보관함에 있는 유물은 장착하지 않아도 약한 패시브 효과를 줄 수 있습니다.

예시:
- 일반 유물: 보관 효과 0.2%~0.5%
- 희귀 유물: 보관 효과 0.5%~1%
- 고대 유물: 보관 효과 1%~2%
- 각성 유물: 보관 효과 2%~3%
- 초월 유물: 보관 효과 3%~5%

단, 보관함 효과가 무한히 중첩되면 밸런스가 무너질 수 있으므로, 같은 종류의 효과는 최대 적용치를 둡니다.

예시:
- 메인게임 포인트 증가 보관 효과 최대 +20%
- 운명권 획득률 증가 보관 효과 최대 +10%
- 예언권 획득률 증가 보관 효과 최대 +10%
- 강화 비용 감소 최대 -15%

# 13. 유물 획득 방식

유물은 다음 방식으로 획득할 수 있습니다.

1. 메인게임 성공 시 낮은 확률로 획득
2. 연속 정답 보상으로 획득
3. 일일 미션 완료로 획득
4. 포인트를 사용한 유물 뽑기
5. 이벤트 보상
6. 특정 업적 보상

## 메인게임 성공 시 유물 획득 확률

초기 추천값:
- 일반 유물: 3%
- 희귀 유물: 0.8%
- 고대 유물: 0.15%
- 각성 유물: 0.03%
- 초월 유물: 직접 드랍 비추천, 승급 중심 권장

초월 유물은 직접 드랍보다는 장기 성장과 승급을 통해 도달하는 방식이 좋습니다.

# 14. 포인트 사용처

포인트 사용처는 다음과 같이 구성합니다.

1. 유물 강화
2. 유물 레벨업
3. 유물 승급
4. 유물 각성
5. 보관함 확장
6. 서브게임 베팅
7. 유물 뽑기
8. 추후 상점 구매

포인트가 계속 쌓이기만 하지 않도록 반드시 소모처를 여러 개 둡니다.

# 15. 전체 플레이 루프

최종 플레이 루프는 다음과 같습니다.

1. 유저가 메인게임을 플레이한다.
2. 정답을 맞히면 포인트를 얻는다.
3. 낮은 확률로 운명권, 예언권, 유물, 유물 재료를 얻는다.
4. 운명권 또는 예언권이 있으면 서브게임에 도전할 수 있다.
5. 서브게임에서 포인트를 늘리거나 잃는다.
6. 획득한 포인트로 유물을 강화/승급/각성한다.
7. 유물 효과로 메인게임 보상과 도전권 획득률이 상승한다.
8. 유저는 더 많은 포인트와 유물을 얻기 위해 다시 메인게임을 플레이한다.

즉, 구조는 다음과 같습니다.

메인게임 → 포인트 획득 → 도전권/유물 획득 → 서브게임 도전 → 포인트 증감 → 유물 성장 → 메인게임 보상 증가 → 반복

# 19. 밸런스 관련 주의사항

아래 사항은 반드시 고려해주세요.

1. 서브게임은 포인트를 무한 생산하는 구조가 되면 안 됩니다.
2. 홀짝과 하이로우는 반드시 도전권이 있어야 플레이 가능해야 합니다.
3. 유물 효과는 성공 확률 직접 증가보다 보상 증가, 손실 완화, 도전권 획득률 증가 중심으로 설계합니다.
4. 보관함 효과는 중첩 상한을 둡니다.
5. 초월 유물은 직접 획득보다 장기 성장으로 도달하게 하는 편이 좋습니다.
6. 신규 유저가 너무 복잡하게 느끼지 않도록, 처음에는 자모연성/홀짝/기본 유물 보관함부터 열고, 승급/각성은 이후 확장해도 됩니다.
7. 모든 수치는 하드코딩하지 말고 설정값으로 분리해주세요.

# 20. 최종 목표

이번 개편의 최종 목표는 초능력자를 단순 퀴즈봇이 아니라 “한글 기반 수집형 성장 게임 봇”으로 만드는 것입니다.

기존:
초성퀴즈/훈민정음 → 포인트 획득 → 끝

변경 후:
메인게임 플레이 → 포인트 획득 → 도전권/유물 획득 → 서브게임 도전 → 포인트 증감 → 유물 강화/승급/각성 → 메인게임 보상 증가 → 반복

이 구조에 맞춰 기존 시스템에서 버릴 부분, 유지할 부분, 새로 추가할 부분을 구분하여 개발 작업 단위로 쪼개주세요.
