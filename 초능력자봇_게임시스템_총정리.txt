1. 포인트(P) 기본 구조
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- 화폐 단위: P (포인트). 게임 정답, 출석, 배틀 등으로 획득.
- 소비처: 유물 뽑기·강화·합성, 닉네임 변경, 보관함 확장, 유물 분해 환급 수령 등.
- 유저별 누적 포인트는 DB의 users.score 컬럼에 저장.
- computeReward() 함수로 최종 보상 계산:
    최종 보상 = 기본 포인트 × (1 + 유물 효과% / 100) + 콤보 보너스
  ※ 유물이 없으면 기본 포인트 그대로 지급.

2. 초성퀴즈
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[개요]
초성(자음)만 보여주고 단어를 맞히는 1인 퀴즈 게임.
카카오톡 방에서 누구나 참여 가능. 방에 동시에 여러 세션 불가.

[명령어]
- 초성퀴즈 / 퀴즈 / ㅊㅅㅋㅈ / 시작 → 랜덤 주제 게임 시작
- 초성퀴즈 [카테고리] → 특정 주제 게임 시작 (예: 초성퀴즈 가수)
- 힌트 → 글자 1개 공개 (100P 차감)
- 다음문제 / 종료

[포인트]
- 랜덤 주제: 최대 6,240P
- 주제 선택: 최대 3,120P
- 오답마다 40~100P 랜덤 감소 (최솟값 보장)
- 힌트 사용 시 100P 추가 차감
- 5회 이상 오답 시 힌트 자동 공개

[규칙]
- 제한시간: 3분 (초과 시 자동 종료, 정답 공개)
- 오답 횟수 제한 없음
- 유물 효과(초성 보너스 + 전체 보너스 + 콤보 보너스) 적용 후 최종 지급

3. 훈민정음 (멀티플레이)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[개요]
초성 + 글자 수 힌트로 단어를 맞히는 멀티플레이 게임.
방 전체가 함께 경쟁하며, 45초 동안 최대한 많은 단어를 맞힘.

[명령어]
- 훈민정음 → 방 개설 (25초 등록 대기)
- 참여 → 게임 참여 등록
- 시작 → 방장이 등록 마감 후 강제 시작

[타임라인]
1. "훈민정음" 입력 → 방 개설
2. 25초 동안 "참여" 입력한 유저 등록
3. 45초 동안 게임 진행 (단어 맞히기)
4. 종료 후 MVP 결정 및 보너스 지급

[포인트]
- 단어 1개 정답: 900P (유물 효과 적용 후 지급)
- MVP 보너스: +5,520P (가장 많이 맞힌 유저 1명)
  ※ MVP 보너스는 다른 참가자가 1개 이상 맞혔을 때만 지급 (몰아주기 방지)
- 콤보 발동 중이면 방 전원 +20% 추가

[특이사항]
- 맞힌 단어는 재사용 불가 (중복 정답 처리 없음)
- 방 내 모든 유저가 공동으로 단어를 맞힘 → 경쟁 + 협력 구조

4. 자모연성
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[개요]
한글 단어를 자모(자음+모음)로 분해해서 보여주고,
4지선다 중 맞는 단어를 고르는 1인 퀴즈 게임.
1, 2, 3, 4 중 하나를 입력.

[명령어]
- 자모연성 → 랜덤 난이도로 시작
- 자모연성 쉬움 / 보통 / 어려움
- 다음퀴즈 → 연속으로 다음 문제
- 자모포기 → 현재 문제 포기

[난이도 및 기본 보상]
- 쉬움  (2글자): 기본 1,780P
- 보통  (3글자): 기본 2,670P
- 어려움 (4글자): 기본 4,000P

[추가 보너스 (중복 적용 가능)]
- 3글자 이상 단어:    +670P
- 받침 포함 단어:     +440P
- 오늘 첫 자모연성:  +1,110P
- 연속 3회 정답마다: +2,220P

[예시: 최대 획득 가능]
  어려움 4글자 + 받침 + 첫자모연성 + 연속3회
  = 4,000 + 440 + 1,110 + 2,220 = 7,770P (유물 효과 적용 전)

[제한시간]
- 3분 (초과 시 자동 종료)

6. 유물 배틀
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[개요]
메인 유물의 전투력으로 다른 유저에게 1:1 도전.
확률 기반 승패 결정 (전투력 차이가 클수록 유리하나 역전 가능).

[명령어]
- 배틀 → 현황 및 남은 횟수 확인
- 배틀 [상대닉네임] → 해당 유저에게 배틀 도전

[전투력 계산 (메인 유물 기준)]
전투력 = 1.0162 × 등급 + 0.03 × 강화단계 + 0.8704 × effect_value × 100

[승패 확률]
sigmoid 함수 기반:
  승리 확률 = sigmoid((내 전투력 - 상대 전투력) / 2000)
  ※ 약자(내가 낮을 때) 내 전투력 ×1.58 보정
  ※ 강자(내가 높을 때) 내 전투력 ×0.55 보정 (역전 방지 완화)

[보상]
- 승리: 상대 메인 유물 전투력(원시값) × 5 P 지급
  강한 상대를 이길수록 더 많은 보상
- 패배: 포인트 차감 없음

[제한]
- 하루 최대 15회 (자정 초기화)
- 자기 자신에게 배틀 불가

7. 유물 시스템
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[등급] D < C < B < A < S (5등급)
[타입] 8종 (나침반/모래시계/랜턴/거울/보물함/왕관/물약/깃털펜)
[명령어] 내유물 / 유물뽑기 / 유물강화 / 유물합성 / 유물분해 / 유물보관함

── 7-1. 유물뽑기 ──────────────────────────────────
- 비용: 1,000P / 1회
- 결과: 랜덤 타입, 확률은 다음과 같음:
    D등급: 4.4121%
    C등급: 3.5000%
    B등급: 0.4788%
    A등급: 0.0833%
    S등급: 0% (뽑기로 획득 불가, 합성으로만 생성)
  ※ 나머지 약 91.5%는 꽝 (아무것도 없음)
- 뽑기 성공 시 기본 D등급으로 획득

── 7-2. 유물 강화 ─────────────────────────────────
- 최대 +20강
- 강화 성공률 (0강 → +1 ~ +19 → +20):
    0강→1강: 95.00%
    1강→2강: 93.00%
    2강→3강: 90.00%
    3강→4강: 84.40%
    4강→5강: 79.25%
    5강→6강: 73.08%
    6강→7강: 67.93%
    7강→8강: 61.76%
    8강→9강: 45.52%
    9강→10강: 40.56%
    10강→11강: 36.42%
    11강→12강: 31.45%
    12강→13강: 27.31%
    13강→14강: 22.35%
    14강~17강: 18.21%
    18강→19강: 14.20%
    19강→20강: 8.52%
- 강화 비용 공식:
    비용 = floor(등급 × 710 × 1.1166^현재강화단계)
    (등급: D=1, C=2, B=3, A=4, S=5)
- 강화 실패 시 등급 유지, 강화단계 변화 없음 (파괴 없음)

── 7-3. 레벨업 ────────────────────────────────────
- 최대 Lv.30
- 레벨업 비용: 등급 × LEVEL_UP_COST_FACTOR × 현재레벨
- 레벨 오를수록 effect_value +1% 보정 (레벨당 +1%)

── 7-4. 효과값(effect_value) 공식 ─────────────────
기본 효과 (등급별, Lv1 +0강 기준):
    D등급: 6.93%
    C등급: 10.56%
    B등급: 17.67%
    A등급: 30.42%
    S등급: 48.45%

최종 효과값 = 기본효과 × (1 + (레벨-1) × 0.01) × (1 + 강화 × 0.03)
  ※ S등급 +20강 Lv30 = 약 100% (설계 기준치)

── 7-5. 유물 합성 ─────────────────────────────────
- 같은 등급 유물 3개 → 상위 등급 1개 (타입 랜덤)
- 합성 비용 (수수료):
    D 3개 → C: 2,123P
    C 3개 → B: 6,521P
    B 3개 → A: 21,685P
    A 3개 → S: 86,890P
- 합성 성공률: 100% (실패 없음)

── 7-6. 유물 분해 ─────────────────────────────────
- 유물을 포인트로 환급
- 기본 환급액:

  + 강화에 사용된 누적 비용 전액 환급

── 7-7. 유물 보관함 ───────────────────────────────
- 메인 유물 외 추가 유물 보관 가능
- 기본 슬롯: 6칸 / 최대 30칸
- 슬롯 확장 비용: 9,000P / 칸
- 보관함 유물 효과 적용 비율: 35.31% (메인 유물의 약 1/3)
- 같은 효과 타입은 최대 3개까지만 스택 (4번째부터 효과 미적용)

── 7-8. 유물 효과 타입 ────────────────────────────
유물 효과는 게임 보상에 직접 적용:

- 강화 비용 절감
- 경험치/레벨업 보너스
- 보관함 효과 증폭 (메타 효과)
- (타입에 따라 적용 효과 다름)

9. 랭킹 시스템 (웹사이트: rank.chosung.app)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

탭 순서: 💎 포인트 (기본) → ⚔️ 유물 → 🌟 통합

── 9-1. 포인트 랭킹 (기본 탭) ─────────────────────
- 정렬: users.score (누적 포인트) 내림차순
- 표시: X,XXX P
- 카카오톡 "랭킹" 명령어도 이 기준 사용
  (방 내 순위 + 전체 상위 5명 + 내 순위 표시)

── 9-2. 유물 랭킹 ─────────────────────────────────
- 정렬: 유물점수 = round(effectSum × 1,400 + relicPower × 15.6)
    effectSum: 메인 + 보관함 전체 유물의 effect_value 합산 (%)
    relicPower: 메인 + 보관함 전체 유물 전투력 합산
               = Σ(1.0162×등급 + 0.03×강화 + 0.8704×effect_value×100)
- 표시: X,XXX P (유물점수)
- 보조 표시: 각 행에 "전투력 X.X · 효과 X.X%" 배지
- 포인트 반영 없음 (유물 강함만 평가)

── 9-3. 통합 랭킹 ─────────────────────────────────
- 정렬: 통합점수 = round(포인트 + effectSum × 1,400 + relicPower × 15.6)
  ※ 통합점수 = 포인트 + 유물점수 (완전 가산)
  ※ 유물 없는 유저: 통합점수 = 포인트 그대로
- 표시: X,XXX 점 (포인트와 구분)
- 배율 근거 (production DB 실측):
    포인트 1위: 약 13,304,591P
    유물 최강 기여분: 약 3,378,241점 (≈ 포인트 1위의 25.4%)
    목표: 유물 기여도 최대 25% 수준으로 설계
- 배율 실시간 조정:
    환경변수: COMBINED_EFFECT_WEIGHT (기본 1400)
              COMBINED_POWER_WEIGHT  (기본 15.6)
    관리자 명령어: "관리 배율 [A] [B]" → 서버 재시작 없이 즉시 변경

[공통 사항]
- 유물 랭킹·통합 랭킹의 A·B 배율이 동일하므로:
    통합점수 = 포인트 + 유물점수  (항등식 성립)
